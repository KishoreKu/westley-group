## ðŸ”§ Customization

### Colors
Edit CSS variables in `index.css` under `:root` to change the color scheme:
```css
--color-primary: hsl(245, 83%, 65%);
--gradient-primary: linear-gradient(135deg, #6366F1 0%, #8B5CF6 100%);
```

### Content
Update text content directly in `index.html` - all sections are clearly labeled with comments.

### Fonts
Change fonts by updating the Google Fonts import in `index.html` and the CSS variables:
```css
--font-primary: 'Inter', sans-serif;
--font-display: 'Space Grotesk', sans-serif;
```
